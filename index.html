SMS Auto-Detection Setup
MacroDroid watches incoming SMSes and hits a webhook in this app automatically. When a payment SMS arrives, the AI popup fires instantly.
  • Install MacroDroid from Play Store (free)
  • Create a new Macro → Trigger: SMS Received → match sender contains GPay OR PhonePe OR Paytm OR HDFC OR ICICI OR SBI
  • Action: HTTP Request → URL: https://your-app-url/?sms={sms_content}
  • Save macro and enable it. Done — payments will now auto-appear in CampusAI!

Use iOS Shortcuts to watch for payment notifications and send them to CampusAI.
  • Open Shortcuts app → tap Automation → New Automation
  • Trigger: App → select GPay / PhonePe / Paytm → "Is Opened"
  • Action: Get Contents of URL → paste your webhook URL below
  • Method: GET, add parameter sms = "Payment detected from app"
  • Turn off "Ask Before Running" → Done!
